Features

Cycle funded Collaborators

Equiped with its cycle fund, the network seeks to reach an unlimited amount of collaborators to build for and on the Nyzo network.

Monitoring

Various management and monitoring tools exist to control large fleets of Nyzo verifiers at once. Access addons for the Nyzo verifier here or view the Nyzo mining farm manager here.

Accessible

Wallets are under constant development for release on new platforms. The Android wallet can be downloaded on the Google Play store here. The iOS version to be released soon. A Windows client can be expected at a later stage of Open Nyzo.

Sender data

Nyzo allows for the inclusion of 32 bits of raw data in every transaction. This enables second-layer solutions to be developed on top of the network.

API

The client comes with various API endpoints. In addition to that, there is a community developed RPC available here.

Full Node

The HistoricalBlockManager allows the client to efficiently access and store historical blocks if needed.
